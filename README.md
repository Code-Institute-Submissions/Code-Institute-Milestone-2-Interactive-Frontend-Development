# Project 2 - Visiting Capital Cities in Europe
This project is built for [Code Institute](https://codeinstitute.net/) as a part of _Full Stack Software Development Diploma course_. Project was focused on using semantic HTML5, SASS along with Bootstrap 4 and JavaScript (jQuery) and working with Foursquer API with using map (leaflet library).

## UX
This is aiming at people who are heading to one of the capital cities in Europe and looking for accommodation, restaurants or nice sights. This website helps them with planning their journey.
